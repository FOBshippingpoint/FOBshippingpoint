# Exclamationer

[Exclamationer](https://fobshippingpoint.github.io/FOBshippingpoint/) 可以幫你把句子轉換成更有魄力的樣子。

## 特色

- **斷詞：** 利用[jieba-js](https://github.com/pulipulichen/jieba-js)進行段詞，把最後一個詞彙的字間都加上驚嘆號
- **自訂：** 更換全形、半形驚嘆號、更改驚嘆號數量
